# Software Engineer Intern (Recommendation Infrastructure) — 2026 Fall (BS/MS)

Steps to get an offer.

---

## Step 1: Online Assessment (GCA) <- on this step right now
The General Coding Assessment is a timed online coding test (60–90 min, 2–4 problems) covering data structures and algorithms. Pass it to move forward.

## Step 2: Technical Interview(s)
One or more live coding interviews focused on problem-solving, algorithms, and system design basics. Expect to explain your thought process clearly.

## Step 3: Team/Behavioral Interview
Discussion about your background, projects, and interest in recommendation infrastructure. Tests communication and culture fit.

## Step 4: Final Review
Recruiters and hiring managers review your performance across all rounds.

## Step 5: Offer
If selected, you'll receive an offer with details on start date, compensation, and team placement.
